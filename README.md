## Getting Started

1. Install dependencies:
```bash
yarn install
```

2. Run development server:
```bash
yarn dev
```

3. Build for production:
```bash
yarn build
```

## Environment Variables

Create a `.env.local` file with:
```env
NEXT_PUBLIC_API_URL=http://localhost:8080
```

## Docker

```bash
docker-compose up -d
```
